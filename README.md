# Dojo
Dojo Engine. Scuffed rectangle rendering engine for **Windows** written in **C++**.

# Current Roadmap

- Sprite rendering implementation
- Entity class

# Logging

Log macros defined:

- Core/Client
  - Trace
  - Info
  - Warn
  - Error
  - Critical

Usage example: DOJO_CORE_ERROR("Error message {0}", variable);
